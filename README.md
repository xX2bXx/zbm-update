# ZBM-Update

[![WTFPL](https://img.shields.io/badge/License-WTFPL-brightgreen.svg)](https://www.wtfpl.net/)

A simple, but perfectly working ZFSBootMenu updater

## Installation
```sh
git clone https://github.com/xX2bXx/zbm-update
cd zbm-update
go build -o zbm-update
```
## Philosophy
- Does one thing well
- No hand-holding
